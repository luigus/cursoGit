# cursoGit

Tutorial git

Iniciando repositorio
$git init

Adicionando arquivo no indice
$git add arquivo

Comitando
$git commit -m "mensagem"

Enviando para servidor remoto
$git push

Criando branch
$git branch develop

Mudando de branch
$git checkout feature

Criando branch git-flow
$git checkout -B feature_branch

Realizando merge
$git merge master
